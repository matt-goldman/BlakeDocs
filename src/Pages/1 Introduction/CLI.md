# CLI Commands

- blake new: Create a new site
- blake bake: Generate site
- blake serve: Serve locally with live reload
- blake init: Add Blake to an existing project

* blake new, --template, --list

* blake bake

* blake serve

* blake init

* (Planned) blake upgrade, blake add-template
